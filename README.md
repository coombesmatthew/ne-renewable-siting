# NE England Renewable Siting & Headroom Map

![Status: under construction](https://img.shields.io/badge/status-under%20construction-orange)

A single-page map of North East England that shows where renewable generation could be developed (resource + grid headroom + low constraints) or acquired (existing REPD pipeline). Frontend is MapLibre GL JS + pmtiles.js reading PMTiles directly from object storage via HTTP range requests; backend is FastAPI on Railway for the site-score and AI chat endpoints; data is precomputed once (no live API hits at runtime). Built as a portfolio demo for a Fuse Energy Operations Manager job application.
